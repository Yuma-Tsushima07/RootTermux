# RootTermux
Root Termux is a way to root be a root user in termux without rooting your phone.

## Install
```
$ apt update
$ apt upgrade
$ apt install git 
$ git clone https://github.com/Yuma-Tsushima07/RootTermux
$ ls
$ cd RootTermux
$ ls
$ chmod 777 root.sh
$ bash root.sh
$ Press Enter (On Keyboard; Don't Write)

(Here Two Options
```
